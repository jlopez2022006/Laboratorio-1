Ciclo Básico

TonysKinal es una aplicación completa diseñada para la gestión integral de un restaurante. Este proyecto surgió como parte de mi formación en el quinto año de perito en informática en Kinal.

La aplicación ofrece diversas funcionalidades, permitiendo a los usuarios realizar diversas acciones como:

  • Realizar pedidos en línea para eventos y servicios a domicilio.
  • Gestionar el inventario de productos y platos disponibles en el menú.
  • Buscar y reservar lugares para eventos especiales.

La aplicación busca facilitar la operación diaria de un restaurante y mejorar la experiencia tanto para los clientes como para el personal.

Diversificado

El Proyecto 2, centrado en la gestión de ventas, es una aplicación completa diseñada para optimizar el proceso de ventas en diferentes contextos comerciales. Este proyecto fue desarrollado como parte de mi formación en el quinto año de perito en informática en Kinal.

Algunas de las características clave de este proyecto incluyen:

  • Gestión integral de clientes, empleados y productos.
  • Registro y seguimiento de transacciones de venta.
  • Capacidad para realizar compras de productos y mantener actualizado el inventario.

La aplicación tiene como objetivo facilitar la administración de las operaciones comerciales y mejorar la eficiencia en el proceso de ventas.
